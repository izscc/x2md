# X2MD v1.1.6

## 更新内容

- 新增 X 博主主页猫爪批量抓取菜单：在博主主页标签栏显示 🐾，可批量抓取原创推文，自动排除转发/转载。
- 支持按设置范围抓取推文：当日、当月、全部、自定义最近 N 天。
- 推文按日期聚合保存：默认写入“保存根路径/博主昵称_handle/博主昵称推文YYYY-MM-DD.md”，并按 X 时间线从新到旧排列。
- 支持 X 博主 Articles 页面批量抓取：页面切到 `/articles` 后，每篇文章保存为单独 Markdown 文件。
- 新增本地抓取记录：同一博主后续抓取会自动跳过已保存的推文/文章；菜单中可选择“重新完整抓取”绕过记录。
- 设置页新增“X 博主批量抓取”：可开关猫爪按钮、配置推文时间范围、自定义批量保存路径。

## 使用方式

1. 打开扩展设置页，在「X 博主批量抓取」中选择时间范围和保存路径。
2. 打开任意 X 博主主页，例如 `https://x.com/xiaoxiaodong01`。
3. 在主页标签栏点击或悬停 🐾，选择“开始抓取”。
4. 若切换到该博主的 Articles 页面，则同一个 🐾 菜单会批量抓取文章，每篇文章单独保存。
