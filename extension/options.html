飞书多维表格设置

▼
在飞书开放平台创建应用获取 App ID/Secret，将应用添加为多维表格的协作者。
表格需包含：标题(文本)、链接(超链接)、作者(文本)、标签(文本)、保存日期(日期)、类型(单选)、正文(文本) 等字段。
📝

Notion Database 设置

▼
需在 Notion Integrations 创建 Integration 获取 Token，然后将 Integration 连接到目标 Database。
属性类型：标题=Title, 链接=URL, 作者=Rich Text, 标签=Multi Select, 保存日期=Date, 类型=Select。
从 Database 链接中复制 32 位 ID
属性映射（必须与 Database 中的列名完全一致）
